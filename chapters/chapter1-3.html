Keyboard/Mouse Commands List (1.3)

These are the keyboard and mouse commands available to you while using the studio editor.

Keyboard

CTRL+ALT+F10*Bring up console
CTRL+SQuick-save map
CTRL+Down arrowZip object
CTRL+Up arrowAllign object to grid without zipping
SpaceEyedrop
F1-F6Change camera height
Numpad numbersRotate camera
Numpad 5Reset camera
CTRL+Z / BackspaceUndo
CTRL+YRedo**
Number keysStudio panels
Numpad +/-Change grid cursor size
DeleteDelete selected object(s)
ESCCancel
TABToggle minimap
RRefresh minimap***
EnterExecute

Mouse

Left clickSelect
Shift+Left clickSelect multiple
Left click+DragSelection box
Right click+DragScroll
F+DragFreelook

*While the editor is open, just F10 will also work, but it's recommended to use CTRL+ALT+F10 anyway. F10 is also used as the keybind to quick-load a mission and you'd hate to do so accidently during gameplay.
**Redo seems unavilable in most if not all situations.
***The minimap has minimal functionality in the studio editor.
