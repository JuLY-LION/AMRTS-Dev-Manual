This section will explain the teams tool - used to create and edit teams - and how to use mission teams.

Creating and editing teams

To create a new team, click "New", then give a name. (If you don't already see it, enable it in the tools menu.) To enter the setup menu for a team, double-click on the team's name.

Name

The name of the team. You should name it based on it's color or purpose if it's for great battle or special operation-type mission.

Color

The color of the team stored as an RGB value. Multiplayer setup overrides this when Available to Players is enabled.

  • The green army uses Color(94, 242, 65); (#5ef241)
  • The tan army uses Color(252, 194, 107); (#fcc26b)
  • The ants use Color(179, 50, 28); (#b3321c)
  • The cockroaches appear to be pre-colored, and won't be affected by their team color

Default Client

Also known as Default Team. If enabled, you'll control this team when the mission is played.

TommyCD1's patched Editor will ensure only 1 team can be set as the default client at a time. Having any other number will cause a crash when the map is played.

Available to Players

Determines if players should be allowed to select this team slot in a multiplayer match. If you have a neutral team that isn't apart of any other team, like pre-placed ants that are hostile to all players, you'll want to disable this option.

You can disregard this for singleplayer missions.

Show in Debriefing

Also called Display Statistics or Has Stats. Enable to show this team's stats during the debriefing.

Requires AI

Used if any units on the team are ever controlled by scripting. This includes the player's units if moved in a cineractive. This has no effect on the personality chosen.

Side

Used to show if the team's army is comprised of army soldiers ("army") or ants/cockroaches. ("insect")

Fixed Side

Enable if team has pre-placed units. Disabling will spawn Sarge and a bulldozer in that team's starting region.

Storage

Determines starting plastic and electricty resources.

Spawn Region

Set this team's spawn region. Click the name above it to center your camera on the current section.

Personality (AI)

The AI configuration to use for this team when not controled by a player. Your default options are None, gb, and default, the last of which is used on Multiplayer.

Camera Angle

Supposedly sets the starting camera angle, but it doesn't have any effect once the mission loads in AMRTS.

Filtering

Determines if the team should have limited access to units or structures. This is enabled for the player team in most of the story campaign, like in mission 4, where choppers are included in the map, (for the Tan) but they're filtered out for the player until mission 7.

Filtering out a unit-type for a team doesn't affect those units already placed on the map. Like mission 8, where the player hasn't unlocked the radio operator, but can still use the one placed on the map for paratroopers should they locate him.
