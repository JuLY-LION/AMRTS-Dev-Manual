This section will explain the teams tool - used to create and edit teams - and how to use mission teams.

Creating and editing teams

To create a new team, click "New", then give a name. (If you don't already see it, enable it in the tools menu.) To enter the setup menu for a team, double-click on the team's name.

Editing a team

The setup menu includes:

Name

The name of the team. You should name it based on it's color or purpose if it's for great battle or special operation-type mission.

Color

The color of the team stored as an RGB value.

Default Client

Also known as Default Team. If enabled, you'll play as this team when the mission is played.

TommyCD1's patched Editor will ensure only 1 team can be set as the default client at a time. Having any other number will cause a crash when the map is played.

Available to Players

Determines if players should be allowed to select this team slot in a multiplayer match.

Show in Debriefing

Add details show this team's stats during the debriefing HasStats Display Statistics

Requires AI

Used if any units on the team are ever controlled by scripting. This includes the player's units if moved in a cineractive.

Side

Used to show if the team's army is comprised of plastic soldiers ("army") or ants. ("insect")

Fixed Side

Enable if team has pre-placed units. Disabling will spawn Sarge and a bulldozer in that team's starting region. FIX THIS IN 4-1

Storage

Determines starting plastic and electricty resources.

Spawn Region

Set this team's spawn region. Click the name above it to center your camera on the current section.

Personality

The AI configuration to use for this team when not controled by a player.
