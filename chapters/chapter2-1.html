Menu Tools

Basic Tools (2.1)

The menu tools are accessed by clicking the "Tools" button in the lower left corner.

New

Creates a new mission from scratch.

For size, enter a single numeric value. Size is measured in meters*. The map will be this size squared, so if you enter 500, the map will be 500x500 meters. Most maps included in AMRTS are around 1024x1024, although that includes unplayable areas, like the kitchen floor in mission 7. For reference:

Camera and Movement/bc_p/T1.x768x768
Behind the Tan Curtain/story/S2.x992x992
Baths of Glory/story/S10.x1024x1024
Flooded Follies/gb/flooded.x1024x1024
DumDum Frenzy/mp/2dumdums.x736x736
Take the 8-Train/mp/8train.x1248x1248
Gehenna Conventions(by TommyCD1)2048x2048

*What was considered a meter in DR2 obviously can't be measured the same in AMRTS, so the proper size is unknown.

The world you select will determine your enviroment tileset and placeable assets. Your default options are sandbox, yard, basement, kitchen, livingrm, bathroom, bedroom, and attic. (Shell is used for non-gameplay screens.) Select one of these worlds in the dropdown box.

Ruleset

This menu is used to set the default ruleset of the map. You probably know it better as a "gamemode". Select a ruleset from the dropdown menu. (AMRTS uses Protect HQ for standard multiplayer maps) You'll want to keep the result on none if you're creating a singleplayer mission, as you'll be either be picking out from the default objectives, or scripting your own later.
